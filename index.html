2. Difference between class and IDs

Ans:- In html, the difference between class and id is that id is unique and each element can have one id.
      On the other hand, classes are not unique, we can use same class on multiple elements.
      Only one ID selector can be attached to an element.
      whereas, multiple class selectors can be attached to an element.   
      In CSS, class can be accessed by using dot (.) 
      whereas, id can be accessed by using hash (#) 
      
    
3. What are self closing tags ?

Ans:- In HTML, self closing tags are the type of HTML tags that need not to be closed manually by its closing tag.
      It means there is no separate closing tag for it .
      Example: img ,input, hr, br

5. Difference between block and inline.

Ans:- In HTML, block element always begins with newline.whereas, inline element doesn't start a new line. 
      Block elements occupy the full width of the screen . Whereas, inline element takes only required width.
      Block level element extends from left to right where browser automatically gives some space(margin) on top and bottom.
      Whereas Inline elements don't have top to bottom margin.
      Example of block level element: div, p , h1 to h6, table, form .
      Example of inline level element: span, img, a

7. What are attributes?

Ans:- In HTML, all HTML element can have attribute. Attributes provide additional inforamtion about element.
      Attributes are always specified in the start tag.
      Attributes usually come in name/value pairs like: name="value"; 
      Example: href, style, src etc
